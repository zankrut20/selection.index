# Changelog

## selection.index 1.2.0

CRAN release: 2023-09-19

- add new function
  [`meanPerformance()`](https://zankrut20.github.io/selection.index/reference/meanPerformance.md)
  for calculating the mean performance in randomized block design data
- remove some typos from the documentation

## selection.index 1.1.4

CRAN release: 2022-06-13

- removed two functions `sel.index()` and `sel.score.rank()` for easy
  implementation of
  [`comb.indices()`](https://zankrut20.github.io/selection.index/reference/comb.indices.md)

## selection.index 1.1.3

CRAN release: 2021-09-25

- Added a new function
  [`gen.advance()`](https://zankrut20.github.io/selection.index/reference/gen.advance.md)
  for genetic advance calculation

## selection.index 1.1.2

CRAN release: 2021-07-26

- rank column added in
  [`rcomb.indices()`](https://zankrut20.github.io/selection.index/reference/rcomb.indices.md)
  for ranking of index

## selection.index 1.1.1

CRAN release: 2021-06-03

- Added a new function
  [`rcomb.indices()`](https://zankrut20.github.io/selection.index/reference/rcomb.indices.md)
  for calculated possible selection indices excluding single character
  or particular index from all possible selection indices

## selection.index 1.1.0

CRAN release: 2021-04-23

- Bug fixes in
  [`phen.varcov()`](https://zankrut20.github.io/selection.index/reference/phen.varcov.md)
- Added a new function
  [`comb.indices()`](https://zankrut20.github.io/selection.index/reference/comb.indices.md)
  for calculating possible selection indices with the group/pairs of
  traits/characters
- removed function: `rank.index()`

## selection.index 1.0.0

CRAN release: 2021-04-06

- Added a `NEWS.md` file to track changes to the package.
