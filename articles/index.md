# Articles

### Articles

- [Getting Started with
  selection.index](https://zankrut20.github.io/selection.index/articles/getting-started.md):

  Quick-start guide for the selection.index R package. Learn to compute
  multi-trait phenotypic selection indices (LPSI) with economic weights,
  genotypic and phenotypic variance-covariance matrices, and rank plant
  breeding genotypes by predicted genetic advance.

- [The Linear Phenotypic Selection Index
  Theory](https://zankrut20.github.io/selection.index/articles/linear-phenotypic-selection-index.md):

  Mathematical theory and R implementation of the Linear Phenotypic
  Selection Index (LPSI). Covers derivation of optimal index
  coefficients, maximized selection response, expected genetic gain per
  trait, and heritability of the index using the selection.index
  package.

- [Constrained Linear Phenotypic Selection
  Indices](https://zankrut20.github.io/selection.index/articles/constrained-linear-phenotypic-selection-indices.md):

  Theory and R implementation of constrained phenotypic selection
  indices: Null Restricted LPSI (RLPSI), Predetermined Proportional
  Gains LPSI (PPG-LPSI), and Desired Gains LPSI (DG-LPSI). Learn to
  impose zero or proportional gain restrictions on plant breeding
  traits.

- [Linear Marker and Genome-Wide Selection
  Indices](https://zankrut20.github.io/selection.index/articles/linear-marker-selection-indices.md):

  Marker-assisted selection in plant breeding using the Linear Marker
  Selection Index (LMSI) and Genome-Wide LMSI (GW-LMSI). Covers
  calculation of marker scores, ridge regression, and combining
  phenotypic and SNP marker data to maximize genetic advance in the
  selection.index R package.

- [Linear Genomic Selection
  Indices](https://zankrut20.github.io/selection.index/articles/linear-genomic-selection-indices.md):

  Linear Genomic Selection Index (LGSI) and Combined LGSI (CLGSI) for
  plant breeding using genomic estimated breeding values (GEBVs). Covers
  mathematical theory, GEBV computation via ridge regression,
  cycle-length efficiency, and R implementation with the selection.index
  package.

- [Constrained Linear Genomic Selection
  Indices](https://zankrut20.github.io/selection.index/articles/constrained-linear-genomic-selection-indices.md):

  Restricted and predetermined proportional gain linear genomic
  selection indices (RLGSI, PPG-LGSI, CRLGSI, CPPG-LGSI) for genomic
  plant breeding. Maximize genetic advance for target traits while
  enforcing null or proportional gain restrictions using GEBVs in the
  selection.index R package.

- [Linear Phenotypic Eigen Selection Index
  Methods](https://zankrut20.github.io/selection.index/articles/linear-phenotypic-eigen-selection-index-methods.md):

  Eigen Selection Index Method (ESIM), Restricted ESIM (RESIM), and
  Predetermined Proportional Gain ESIM (PPG-ESIM) for plant breeding.
  Estimate optimal economic weights using canonical correlation and SVD
  without prior weight specification, implemented in the selection.index
  R package.

- [Linear Genomic Eigen Selection Index
  Methods](https://zankrut20.github.io/selection.index/articles/linear-genomic-eigen-selection-index-methods.md):

  Genomic eigen selection index methods using GEBVs and marker data:
  MESIM, GESIM, GW-ESIM, RGESIM, and PPG-GESIM. Integrate genome-wide
  marker information into eigen-based selection frameworks to maximize
  genetic advance in plant breeding with the selection.index R package.

- [Multistage Phenotypic Selection
  Indices](https://zankrut20.github.io/selection.index/articles/multistage-linear-selection-indices.md):

  Multistage linear selection indices (MLPSI, MRLPSI, MPPG-LPSI, MLGSI,
  MRLGSI, MPPG-LGSI) for sequential multi-environment plant breeding
  programs. Adjust phenotypic and genomic covariance matrices for prior
  selection with the Cochran-Cunningham method using the selection.index
  R package.

- [Stochastic Simulation of Linear Phenotypic Selection
  Indices](https://zankrut20.github.io/selection.index/articles/stochastic-simulation-linear-phenotypic-indices.md):

  Stochastic simulation of recurrent plant breeding selection cycles
  using LPSI, ESIM, RLPSI, and RESIM indices. Evaluate long-term genetic
  gain, variance decay, and selection accuracy over 50+ generations with
  the selection.index R package’s simulate_selection_cycles function.
