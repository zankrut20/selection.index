# Linear Marker and Genome-Wide Selection Indices (Chapter 4)

Implements marker-based selection indices from Chapter 4 (Lande &
Thompson, 1990). These methods incorporate molecular marker information
directly into selection indices.

Methods included: - LMSI : Linear Marker Selection Index (Section 4.1) -
GW-LMSI : Genome-Wide Linear Marker Selection Index (Section 4.2)
