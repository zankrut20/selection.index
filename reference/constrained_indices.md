# Constrained Phenotypic Selection Indices (Chapter 3)

Implements constrained phenotypic selection index methods from Chapter
3. These methods allow breeders to impose restrictions on genetic gains
or specify target gains while maintaining selection efficiency.

Methods included: - Restricted Linear Phenotypic Selection Index
(RLPSI) - Kempthorne & Nordskog (1959) - Predetermined Proportional
Gains (PPG-LPSI) - Tallis (1962) - Desired Gains Index (DG-LPSI) - Pesek
& Baker (1969)

All implementations use C++ primitives for mathematical operations.

## References
