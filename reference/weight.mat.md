# Convert dataframe to matrix

Convert dataframe to matrix

## Usage

``` r
weight.mat(data)
```

## Arguments

- data:

  dataframe of weight

## Value

A matrix

## Examples

``` r
weight.mat(data = weight)
#>      ew     h2
#> [1,]  1 0.6947
#> [2,]  1 0.3244
#> [3,]  1 0.6861
#> [4,]  1 0.7097
#> [5,]  1 0.8336
#> [6,]  1 0.2201
#> [7,]  1 0.5226
```
