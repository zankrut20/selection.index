# Generate Offspring Population Through Recombination

Creates the next generation by randomly mating selected parents and
simulating meiotic recombination.

## Usage

``` r
.generate_offspring(selected_parents, n_offspring)
```

## Arguments

- selected_parents:

  List containing haplotypes and metadata of selected individuals

- n_offspring:

  Number of offspring to generate

## Value

List with offspring haplotypes
