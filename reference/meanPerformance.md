# Mean performance of phenotypic data

Mean performance of phenotypic data

## Usage

``` r
meanPerformance(data, genotypes, replications)
```

## Arguments

- data:

  data for analysis

- genotypes:

  genotypes vector

- replications:

  replication vector

## Value

Dataframe of mean performance analysis
